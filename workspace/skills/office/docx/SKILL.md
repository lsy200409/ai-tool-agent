# Word 文档处理

## 概述
帮助大学生撰写和格式化各类Word文档：课程论文、实验报告、求职简历、申请书等。

## 文档类型与结构

### 课程论文
```
封面 → 摘要+关键词 → 目录 → 引言 → 正文(多章) → 结论 → 参考文献 → 致谢
```
- 摘要200-300字
- 正文小四号宋体，行距1.5倍
- 标题黑体，一级二号，二级三号
- 页边距：上下2.54cm，左右3.18cm

### 实验报告
```
实验名称 → 实验目的 → 实验原理 → 实验器材 → 实验步骤 → 数据记录 → 数据分析 → 结论
```
- 数据部分使用表格呈现
- 图表需编号并有标题（图1、表1）

### 求职简历
- 一页为限
- 教育背景 → 实习/项目经历 → 技能证书 → 自我评价
- 经历使用STAR法则：情境→任务→行动→结果，量化成果

### 申请书
- 标题居中，二号黑体
- 称呼顶格，正文空两格
- 落款右对齐（申请人+日期）

## 格式规范
- 章节自动编号（1 / 1.1 / 1.1.1）
- 图表按章节编号（图1-1、表2-3）
- 参考文献使用GB/T 7714格式
- 页眉页脚：奇数页论文标题，偶数页章节标题

## 输出格式
根据文档类型提供完整的结构大纲和关键格式提醒。
