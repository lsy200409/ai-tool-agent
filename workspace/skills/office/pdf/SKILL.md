# PDF 文档处理

## 概述
帮助大学生处理PDF文档：提取内容、格式转换、合并拆分、阅读批注等。

## 常见操作

### 内容提取
从PDF中提取文字、表格、图片，支持：
- 学术论文的摘要和参考文献提取
- 课件PPT转文字笔记
- 扫描版书籍OCR识别

### 格式转换
- **PDF → Word**：保持原格式排版
- **PDF → Markdown**：适合做读书笔记
- **PDF → 图片**：逐页导出为图片

### 合并与拆分
- 多份实验报告合并为一个PDF
- 从论文集拆出需要的单篇
- 提取指定页码范围

### 阅读辅助
- 标注重点段落
- 提取目录结构
- 搜索关键词并定位

## 注意事项
- 扫描版PDF需OCR处理
- 密码保护的PDF需要先解密
- 含大量图片的PDF文件可能较大

## 输出格式
根据用户需求输出提取的文本、转换后的文件路径或操作结果说明。
